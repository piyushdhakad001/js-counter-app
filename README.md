# Counter App 🔢

A simple counter app built with vanilla JavaScript.
Supports increment, decrement, reset, and saves count using localStorage.

---

## Features
- ➕ Increment count
- ➖ Decrement count
- 🔄 Reset to default (1000)
- 💾 Saves last count using localStorage

---

## Tech Used
- HTML
- CSS
- JavaScript (Vanilla)

---

## How to Run
1. Clone the repo
2. Open `index.html` in your browser
